  • Python-based program
    Expenses Tracker

    The Expenses Tracker is a Python-based program designed to track and visualize personal expenses. Using Tkinter for the graphical user interface (GUI) and Matplotlib for data visualization, the project allows users to input and categorize their expenses while displaying the data in charts. As part of the Python Programming course by IBM, this coding project helped me develop proficiency in Python, enhance my skills in GUI development, and apply data visualization techniques to display financial data effectively. I also used AI tools like ChatGPT to assist in solving coding challenges and optimizing the functionality.

  • Design Thinking project
    Bus Live Location Tracking App

    The Busify app was developed as part of a Design Thinking project to understand and apply the principles of human-centered design. The goal was to create a real-time bus tracking app that enhances user convenience by improving the commuting experience. In this project, we followed the Design Thinking process, which involves empathizing with users, defining problems, ideating solutions, prototyping, and testing. We used FlutterFlow for app development, Figma for UI/UX design, and integrated various AI tools to enhance the app’s features. This project helped us apply Design Thinking principles and create a practical solution focused on user needs.
